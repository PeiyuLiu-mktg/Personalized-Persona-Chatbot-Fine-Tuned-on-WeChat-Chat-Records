# 微信人格聊天系统课程提交材料说明

本目录按课程要求整理为 5 个主体部分，包含代码、数据说明、评测报告、展示材料、训练与评测脚本及团队分工说明。

## 目录对应关系

- `01_代码仓库/`
  - 代码主体与核心脚本。
  - `README.md` 说明代码用途与入口。
- `02_数据说明文档/`
  - 训练集、测试集样本与预处理 notebook。
  - `数据说明与合规说明.md`。
- `03_评测报告/`
  - 课程报告 `tex/pdf` 与评测结果汇总。
- `04_展示Slides/`
  - 课堂展示 PPT。
- `05_训练与评测脚本/`
  - 训练配置、训练日志、评测脚本。
  - `README.md`、`训练脚本/`、`推理脚本/` 中给出复现入口。
