# 数据说明与合规说明

## 1. 数据来源

本项目数据来源于成员本人持有的微信私聊记录导出 JSON。原始消息经筛选与结构化后，整理为监督微调数据集，用于人格化聊天模型训练与离线评测。

## 2. 数据规模

按人物划分后的训练/测试集规模如下：

| 目标人物 | 训练集 | 测试集 | 合计 |
|---|---:|---:|---:|
| xiaolaoshi | 942 | 105 | 1047 |
| lixv | 2613 | 291 | 2904 |
| liupeilin | 2070 | 230 | 2300 |
| 合计 | 5625 | 626 | 6251 |

对应样本文件位于 [数据样本](/D:/xwechat_files/课程提交材料/02_数据说明文档/数据样本)。

## 3. 数据格式

训练与测试样本采用 LLaMA-Factory 可直接读取的 Alpaca 风格 JSON 结构，主要字段包括：

- `instruction`
- `input`
- `output`
- `system`
- `history`

其中 `history` 用于保留有限轮上下文，帮助模型学习连续对话中的语气与风格延续。

## 4. 处理流程

数据处理流程如下：

1. 从微信私聊导出原始 JSON。
2. 仅保留文本消息，过滤图片、文件、语音等非文本内容。
3. 按时间顺序配对“对方输入 - 目标人物回复”。
4. 构造有限轮 `history` 上下文。
5. 划分训练集与测试集。
6. 输出为 LLaMA-Factory 可读的 JSON 文件。

预处理 notebook 见：

- [数据预处理.ipynb](/D:/xwechat_files/课程提交材料/02_数据说明文档/数据样本/数据预处理.ipynb)

## 5. 合规与使用边界

- 数据仅用于课程项目实验与课堂展示，不用于公开发布原始聊天内容。
- 样本中已尽量避免提交过多敏感上下文，提交目录只保留课程复现实验所需数据。
- 若涉及第三方隐私信息，应在公开展示或二次传播前继续脱敏。
- 本项目不主张将原始微信聊天记录直接公开上传至开放社区。
